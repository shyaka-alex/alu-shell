# Shell Basics
Scripts in this directory:
- 0-current_working_directory: prints current working directory
- 1-listit: lists current directory contents
- 2-bring_me_home: changes to home directory
- 3-listfiles: lists files in long format
- 4-listmorefiles: lists files including hidden ones
- 5-listfilesdigitonly: lists files with numeric user/group IDs
- 6-firstdirectory: creates a directory in /tmp
- 7-movethatfile: moves betty file
- 8-firstdelete: deletes betty file
- 9-firstdirdeletion: deletes my_first_directory
- 10-back: goes to previous directory
- 11-lists: lists current, parent and /boot directories
- 12-file_type: prints file type of iamafile
- 13-symbolic_link: creates symbolic link to /bin/ls
- 14-copy_html: copies HTML files to parent directory
- 15-lets_move: moves uppercase files to /tmp/u
- 16-clean_emacs: deletes files ending with ~
- 17-tree: creates directory tree
