# Shell Loops, Conditions and Parsing

- 1-for_best_school: displays Best School 10 times using for loop
- 2-while_best_school: displays Best School 10 times using while loop
- 3-until_best_school: displays Best School 10 times using until loop
- 4-if_9_say_hi: displays Best School 10 times with Hi on 9th iteration
- 5-4_bad_luck_8_is_your_chance: bad luck on 4th, good luck on 8th
- 6-superstitious_numbers: displays numbers 1-20 with superstitious messages
- 7-clock: displays time for 12 hours and 59 minutes
- 8-for_ls: displays directory contents after first dash
- 9-to_file_or_not_to_file: gives info about school file
- 10-fizzbuzz: displays FizzBuzz from 1 to 100
- 11-read_and_cut: displays username, uid and home from /etc/passwd
- 12-tell_the_story_of_passwd: displays /etc/passwd as a story
- 13-lets_parse_apache_logs: displays IP and HTTP code from Apache log
- 14-dig_the-data: groups visitors by IP and HTTP status code
