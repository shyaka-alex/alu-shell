# Shell Init Files, Variables and Expansions

- 0-alias: creates alias ls=rm *
- 1-hello_you: prints hello user
- 2-path: adds /action to PATH
- 3-paths: counts directories in PATH
- 4-global_variables: lists environment variables
- 5-local_variables: lists all local variables and functions
- 6-create_local_variable: creates local variable BEST=School
- 7-create_global_variable: creates global variable BEST=School
- 8-true_knowledge: adds 128 to TRUEKNOWLEDGE
- 9-divide_and_rule: divides POWER by DIVIDE
- 10-love_exponent_breath: raises BREATH to power LOVE
- 11-binary_to_decimal: converts binary to decimal
- 12-combinations: prints all two-letter combinations except oo
- 13-print_float: prints number with 2 decimal places
- 14-decimal_to_hexadecimal: converts decimal to hex
- 15-rot13: encodes/decodes text using rot13
- 16-odd: prints every other line starting with first
- 17-water_and_stir: adds WATER and STIR in custom bases
